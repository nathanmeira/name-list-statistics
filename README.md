# Name List Statistics

Find out the number of masculine and feminine names in a list.

### Stack:

- Nuxt.JS
- TailwindCSS
- JavaScript

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
```
